# Changelog

All notable changes to MACROFORGE will be documented here.

## [1.0.0] - 2025

### Initial Release
- Global hotkey system — one key runs the currently selected macro
- Group/folder system for organizing macros by game or category
- Key press steps with configurable delay and repeat count
- Mouse click steps with coordinate picker
- Multi-step sequences with reordering
- Loop mode with configurable loop delay
- High-accuracy timing using Windows 1ms timer resolution
- Auto-save on exit, auto-load on launch
- Standalone .exe build via PyInstaller
- Custom MACROFORGE icon
